Coluna Descrição
Origem arquivo Indica em qual Sistema o arquivo é gerado.
Obrigatoriedade Arquivos OBRIGATÓRIOS são necessários para o cálculo diário. Arquivos IMPORTANTES são utilizados em outras funcionalidades do sistema.
Situação importação ATUALIZADO: arquivo já foi importado corretamente hoje | ATRASADO: arquivo diário: não foi importado há, pelo menos, 1 dia. Mensal: não importado há 1 mês | IMPORTANDO: arquivo está sendo importado para o Navega | AGUARDANDO: arquivo está na.
Periodicidade importação Período de tempo em que o arquivo deve ser importado: diário, mensal ou eventual (somente em caso de necessidade).
Descrição do arquivo Descrição do que se refere o arquivo.
Nome padrão do arquivo Nome do arquivo para tratamento do Navega.
Tipo entrada Forma como o arquivo entrou no Navega: MANUAL: importado pela tela do Navega manualmente | PASTA: colocado na pasta compartilhada.
Arquivo importado Nome do arquivo importado pelo Navega, considerando que, quando a importação é manual, não é obrigatório utilizar o nome padrão.
Início da importação Data e hora que o Navega iniciou a importação do arquivo (processamento) e não a hora que foi colocado na pasta.
Fim da importação Data e hora que a importação do arquivo foi concluída.
Tempo gasto Tempo gasto na importação do arquivo do início ao fim.
Data de referência Maior data de referência dos dados contidos no arquivo. No caso das tabelas que contém saldos e valores, se refere à data dos saldos ou data de movimento.
